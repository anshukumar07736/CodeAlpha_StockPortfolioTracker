# CodeAlpha_StockPortfolioTracker
# Stock Portfolio Tracker using Python

## Aim
To develop a simple stock portfolio tracker that calculates the total investment value.

## Description
This project is a Stock Portfolio Tracker developed using Python.
The user enters stock names and quantities, and the program calculates
the total investment based on predefined stock prices.
The final investment summary is displayed on the screen and saved to a text file.

## Concepts Used
- Python Dictionary
- Loops
- Conditional Statements
- User Input / Output
- File Handling

## How to Run
1. Install Python
2. Run stock_tracker.py
3. Enter stock names and quantities
4. Type 'done' to finish

## Future Scope
This project can be enhanced by adding real-time stock prices using APIs
and storing data in a database.
